###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `{}`

<i>This line declares a variable named "greeting" using the let keyword. However, it does not assign any value to it at this point. Essentially, it creates a variable that is undefined.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: `"12"`

<i>The result of "1" + "2" is the string "12" because it concatenates the two strings together.
So, the result of calling sum(1, "2") is "12", which is a string, not the numeric value 3 that you might expect from numerical addition</i>

</p>
</details>
